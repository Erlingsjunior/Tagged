# Arquivos de Som

## bloop.mp3

Este arquivo precisa ser adicionado manualmente. O som deve ser um "bloop" curto e agradável que toca quando o usuário dá like em uma denúncia.

### Sugestões:

-   Duração: 200-400ms
-   Tom: Alegre e leve
-   Volume: Moderado

### Onde encontrar sons:

-   https://freesound.org/
-   https://mixkit.co/free-sound-effects/
-   https://www.zapsplat.com/

### Exemplo de busca:

Procure por: "bloop", "pop", "notification", "like", "heart"

### Formato:

-   Arquivo: MP3
-   Nome: `bloop.mp3`
-   Localização: `assets/sounds/bloop.mp3`
